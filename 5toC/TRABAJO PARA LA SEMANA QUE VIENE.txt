ACLARACION: para ambos ejercicios, suponer que el usuario siempre ingresa un número válido, es decir, no ingresa texto como "hola" ni nada raro.



1) Crear un programa en Python que solicite al usuario un número entero y muestre su tabla pitagórica.
Cada línea de la tabla pitagórica debe mostrarse en el siguiente formato: n x m = p, donde n es el número ingresado por el usuario, m es el multiplicador (variando desde 0 hasta 10), y p es el producto de n y m.

Por ejemplo, si el usuario ingresa el número 3, la salida del programa debería ser:

3 x 0 = 0
3 x 1 = 3
3 x 2 = 6
3 x 3 = 9
3 x 4 = 12
3 x 5 = 15
3 x 6 = 18
3 x 7 = 21
3 x 8 = 24
3 x 9 = 27
3 x 10 = 30





2) Crear un programa en Python que solicite al usuario que ingrese números hasta que se ingrese un 0, luego muestre la suma de todos los números ingresados hasta el momento, y finalmente indique si dicha suma es mayor o menor a 100:

Por ejemplo, si el usuario ingresa los números 50, 60, 0, la salida del programa debería ser:

La suma total de los números ingresados es 110
La suma es mayor a 100
